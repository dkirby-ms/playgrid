   - Determined GITHUB_TOKEN already prevents recursive workflow triggers (built-in GitHub behavior)
   - Directive was redundant and dangerous in squash-merge context

## Decision Rationale

- GITHUB_TOKEN prevents workflow recursion without `[skip ci]`

- Deploy reliability: UAT/prod deploys will no longer be silently skipped
