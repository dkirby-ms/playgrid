## Issue

Risk game deadlocked in setup phase on UAT (reported by dkirby-ms). Last player placing armies would finish, but turn would wrap to an already-finished player with no valid moves to trigger `endPhase`. Game stuck forever.

## Resolution

**Gately:** Added global `allDone` check in `placeArmy` handler to auto-transition setup → playing phase.

**Steeply:** Wrote 14 regression tests covering 2/3/6 player variants and edge cases.

**Hal:** Reviewed PR #144, approved, and codified team pattern for global phase transitions.
