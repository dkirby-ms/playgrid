## Key Decisions

1. **Generic system:** Chess clock logic in BaseGameRoom (not CheckersPlugin), per Copilot directive
2. **Schema in BaseGameState:** player1TimeRemainingMs, player2TimeRemainingMs (default 0)
3. **Configuration interface:** ChessClockConfiguration in IGamePlugin (enabled, initialTimeBankMs)
4. **MVP hardcoded values:** 10-minute banks; UI toggle deferred to P7+
5. **UI design:** Sidebar panel (4th position) + per-player info bar times; all colors from existing tokens
