# playgrid

> Play classic games with friends

Built with the [Eschaton Studio](https://github.com/dkirby-ms/eschaton-studio) game development framework.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Server | Colyseus (WebSocket game server) |
| Client | PixiJS + Vite (HTML5 Canvas) |
| Shared | Pure TypeScript types and constants |
| Testing | Vitest |

## Getting Started

```bash
npm install
npm run dev
```
